# Manual Context Curation Is Hidden Labor
*â–³ problem*

Both technical and non-technical users are doing the same invisible job: manually curating context for AI. They copy-paste from old chats, reopen long threads, search through files, re-explain background repeatedly, and manually attach documents. This labor is invisible because no tool measures or surfaces it.

## Why it matters

Hidden labor means hidden opportunity. Whoever makes this labor visible and then eliminates it captures an enormous workflow improvement that users feel immediately.
